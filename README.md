# Superstore Sales Analytics
A comprehensive data analysis project exploring global superstore sales data. This project analyzes sales patterns, profitability trends, and performance metrics across different regions, customer segments, and product categories using Python and Jupyter Notebook.

## Key Features:

- Sales and profit analysis across multiple markets and regions
- Customer segmentation analysis
- Product category and sub-category performance tracking
- Monthly sales trend analysis
- Top performing products and customers identification
- Regional performance comparison
- Discount impact assessment on profitability
- Interactive visualizations and statistical summaries
* Dataset: *  51,000+ global superstore orders spanning multiple years with detailed transaction information including sales, profit, discount, shipping costs, and geographic data.

* Tech Stack: * Python, Pandas, Matplotlib, Jupyter Notebook
